account_create

このページは Vercel 用の静的ランディングです。リポジトリのメインアプリは agent2/ の FastAPI + uvicorn（ローカル API・長時間の Claude 呼び出し・サブプロセス）のため、 そのままでは Vercel のサーバーレスだけでは同じ UI は動きません（404 やタイムアウトの原因になります）。

ローカルでデモ UI を使う:
cd agent2 → uvicorn demo.server:app --host 127.0.0.1 --port 8765 → ブラウザで http://127.0.0.1:8765

クラウドに載せる場合は、Railway / Render / Fly.io など「常時起動のコンテナ」向けです。リポジトリの agent2/Dockerfile を参照してください。

ドキュメント: リポジトリの README.md と agent2/README.md
